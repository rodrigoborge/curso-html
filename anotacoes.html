    </picture>

CSS

Páginas Únicas

Para incluir os estilos CSS o ideal - para páginas únicas - é inserir a tag logo abaixo do título, antes de fechar a seção head

A tag é aberta com <style> e abaixo dela é declarado o nome da seção e é aberta uma chave {, com isso começamos a digitar os valores dos estilos, como por exemplo:

    
    <style>
    body{
        background-color: darkturquoise;
        font-family: sans-serif;
        font-size: 20px;
    }
    h1{
        color: dodgerblue;
        background-color: ghostwhite;
    }
    h2{
        color: forestgreen;
    }
    p{
        text-align: justify;
    }
    </style>

Várias Páginas

Usar a tag link:css, clicar em cima do link de página criado e criar uma nova página com base nele. A tag ficará assim 
        
    <link rel="stylesheet" href="style.css">

Com isso já podemos inserir os estilos dentro da folha de estilos e ela passará a valer para todas as páginas que contenham a referência.

Para mexer nas configurações globais do arquivo, é utilizado a tag asterisco *.

Para mexer no corpo do arquivo é usada a tag body dentro da sessão head.

Toda a tag em css é seguida de chave {

Cores Gradientes

Para usar as cores com gradientes no CSS é usada a tag background-image: linear-gradient, em seguida é dada a direção que o gradiente terá, como por exemplo to bottom para ir para o final da página, e em seguida são indicadas as cores, separadas por virula, sem limite de cores.

Exemplo de um fundo transparente para o corpo da página:


    body{
        background-image: linear-gradient(to bottom, #F24495, #181926);
        background-attachment: fixed;
    }

Fontes em CSS

Para alterar a fonte no arquivo CSS é usado o comando font-family e selecionada a família de fontes que mais combina com o projeto.

Caso tenha dúvida sobre as fontes que combinam entre si, é possível consultar o CSS Web Safe Font

Ou ainda, utilizar após digitar o comando font-family: as teclas ctrl+espaço para que o VS Code sugira algumas famílias de fonte, como no exemplo abaixo


    body{
            font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;
        }

No exemplo a família utilizada é a -apple-system

Tamanho de Fontes

Medidas Absolutas:

  • Centímetro → cm
  • Milímetro → mm
  • Polegada → in
  • Pixel → px
  • Ponto → pt
  • Paica → pc

Nestes casos as apresentações podem ser diferentes. Até a medida de pixel, pois dependendo do dispositivo o tamanho dos pixels variam, como por exemplo as telas retinas da Apple, onde 1 pixel não é 1 pixel, mas sim 1/4 de pixel.

Medidas Relativas:

  • em → Relativo ao tamanho natural da fonte, que no caso são 16px o padrão
  • ex → Relativo a altura X da fonte
  • rem → Relativo a fonte configurada no body
  • vw → Relativo a largura da view port
  • vh → Relativo a altura da viel port
  • % → Relativo a % inserida

Para medida de fonte, é recomendado o uso do px ou em.

16px geralmente é igual a 1em

Exemplo: Se 16px é o tamanho natural da fonte no navegador e é representado por 1em, 2em é igual a 32px

Peso das Fontes

Para determinar o peso da font é usado o comando font-weight, e é possível trabalhar as fontes com os seguintes pesos

  • lighter
  • normal
  • bold
  • bolder

Ainda é possível trabalhar com as opções numéricas, que começam em 100 e terminam em 900, ficando assim:

  • 100 → lighter
  • 400 → normal
  • 600 → bold
  • 800 → bolder

Itálico no CSS

Para deixar a fonte em itálico pelo CSS é preciso usar o comando font-style e usar: italic.

Sublinhado no CSS

Para usar o sublinhado no CSS é preciso usar o comando text-decoration e usar: underline

Usando o Google Fonts

Para importar as fontes do Google Fonts, basta selecionar a fonte desejada, clicar no cubo e copiar o @import, para declarar a fonte é preciso copiar o campo de font-family indicada pelo Google.

O import precisa ser logo abaixo da tag style, fora do body ou de qualquer outro seletor.

Fontes Externas

Para adicionar fontes externas é utilizado o comando @font-face dentro da tag style, este comando chama a fonte salva na pasta do servidor. Desta forma, ficará assim: 

    
    @font-face {
        font-family: 'Chicken';
        src: url(fontes/ChickenSoup.otf); format('opentype'), url(fontes/ChickenSoup.otf) format('truetype')

    Modelo de Caixas - Box Model

    Anatomia de uma Caixa

    • Height: Altura
    • Width: Largura
    • Border: Borda
    • Padding: Preenchimento do espaço interno
    • Margin: Preenchimento espaço externo
    • Outline: Contorno além da borda

    Tipos de Caixa

    • box-level: Se inicia sempre em uma linha nova e ocupa sempre 100% da area do navegador
      • Exemplos
        • div
        • h1-h6
        • p
        • main
        • header
        • nav
        • article
        • aside
        • footer
        • form
        • video
    • inline-level: Fica sempre na mesma linha e dentro da linha, não quebra linha
      • Exemplo:
        • span
        • a
        • code
        • small
        • strong
        • em
        • sup - sub
        • label
        • button
        • input
        • select
